# Repository Context

This is a **Neovim configuration repository** located at `~/.config/nvim/`.

## Key Information
- **Type**: Personal Neovim configuration
- **Language**: Lua-based configuration
- **Minimum Version**: Neovim 0.11 or greater
- **Purpose**: Customized text editor setup for Neovim
- **Main entry point**: `init.lua`
- **Structure**: 
  - `lua/` - Lua configuration modules
  - `plugin/` - Plugin configurations
  - `lsp/` - Language Server Protocol settings
  - `pack/` - Package manager files

When working with this repository, you are modifying Neovim editor configuration files.
